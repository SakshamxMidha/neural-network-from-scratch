# Neural Network From Scratch 🧠

## 📌 Overview

This project implements a **feedforward neural network from scratch using NumPy** for binary classification on a non-linear dataset.
It demonstrates core deep learning concepts without using high-level frameworks like TensorFlow or PyTorch.

---

## 🚀 Features

* Implementation of neural network using only **NumPy**
* Manual **forward propagation**
* Manual **backpropagation**
* **Gradient descent** optimization
* Training & validation **loss tracking**
* Visualization of model performance using **Matplotlib**

---

## 🧠 Concepts Covered

* Neural Network Architecture (Input → Hidden → Output)
* Activation Functions
* Loss Functions
* Backpropagation Algorithm
* Gradient Descent Optimization

---

## 📊 Dataset

* Uses **non-linear synthetic dataset** (`make_moons` from Scikit-learn)
* Suitable for testing classification performance of neural networks

---

## ⚙️ Tech Stack

* Python 🐍
* NumPy
* Matplotlib
* Scikit-learn

---

## 📈 Output

* Model learns to classify non-linear data
* Loss vs Epoch graph shows training progress
